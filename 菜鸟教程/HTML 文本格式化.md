HTML 包含多个用于定义具有特殊含义的文本的元素。

## 实例

```html
这段文本是粗体

这段文本是斜体

这是下标和上标
```

## HTML 格式化元素

格式化元素用于显示特殊类型的文本：

- `<b>` - 粗体文本
- `<strong>` - 重要文本
- `<i>` - 斜体文本
- `<em>` - 强调文本
- `<mark>` - 标记文本
- `<small>` - 较小文本
- `<del>` - 删除文本
- `<ins>` - 插入文本
- `<sub>` - 下标文本
- `<sup>` - 上标文本

## HTML <b> 和 <strong> 元素

HTML `<b>` 元素定义粗体文本，不带任何额外的重要性。

### 实例

```html
<b>这段文本是粗体</b>
```

[亲自试一试](https://www.w3school.com.cn/tiy/t.asp?f=html_formatting_b)

HTML `<strong>` 元素定义具有强烈重要性的文本。其内容通常以粗体显示。

### 实例

```html
<strong>这段文本很重要！</strong>
```

[亲自试一试](https://www.w3school.com.cn/tiy/t.asp?f=html_formatting_strong)

## HTML <i> 和 <em> 元素

HTML `<i>` 元素定义以替代声音或语气呈现的部分文本。其内容通常以斜体显示。

提示： `<i>` 标签常用于表示技术术语、外语短语、想法、船名等。

### 实例

```html
<i>这段文本是斜体</i>
```

[亲自试一试](https://www.w3school.com.cn/tiy/t.asp?f=html_formatting_i)

HTML `<em>` 元素定义强调文本。其内容通常以斜体显示。

提示：屏幕阅读器会以强调的语气读出 `<em>` 中的词语。

### 实例

```html
<em>这段文本被强调</em>
```

[亲自试一试](https://www.w3school.com.cn/tiy/t.asp?f=html_formatting_em)

## HTML <small> 元素

HTML `<small>` 元素定义较小的文本：

### 实例

```html
<small>这是一些较小的文本。</small>
```

[亲自试一试](https://www.w3school.com.cn/tiy/t.asp?f=html_formatting_small)

## HTML <mark> 元素

HTML `<mark>` 元素定义应被标记或高亮显示的文本：

### 实例

```html
<p>今天别忘了买 <mark>牛奶</mark>。</p>
```

[亲自试一试](https://www.w3school.com.cn/tiy/t.asp?f=html_formatting_mark)

## HTML <del> 元素

HTML `<del>` 元素定义已从文档中删除的文本。浏览器通常会在删除的文本上添加删除线：

### 实例

```html
<p>我最喜欢的颜色是 <del>蓝色</del> 红色。</p>
```

[亲自试一试](https://www.w3school.com.cn/tiy/t.asp?f=html_formatting_del)

## HTML <ins> 元素

HTML `<ins>` 元素定义已插入文档的文本。浏览器通常会在插入的文本下添加下划线：

### 实例

```html
<p>我最喜欢的颜色是 <del>蓝色</del> <ins>红色</ins>。</p>
```

[亲自试一试](https://www.w3school.com.cn/tiy/t.asp?f=html_formatting_del_ins)

## HTML <sub> 元素

HTML `<sub>` 元素定义下标文本。下标文本出现在正常行下方半个字符的位置，有时以较小的字体呈现。下标文本可用于化学式，如 H <sub>2</sub> O：

### 实例

```html
<p>这是 <sub>下标</sub> 文本。</p>
```

[亲自试一试](https://www.w3school.com.cn/tiy/t.asp?f=html_formatting_sub)

## HTML <sup> 元素

HTML `<sup>` 元素定义上标文本。上标文本出现在正常行上方半个字符的位置，有时以较小的字体呈现。上标文本可用于脚注，如 WWW <sup>[1]</sup> ：

### 实例

```html
<p>这是 <sup>上标</sup> 文本。</p>
```

[亲自试一试](https://www.w3school.com.cn/tiy/t.asp?f=html_formatting_sup)
