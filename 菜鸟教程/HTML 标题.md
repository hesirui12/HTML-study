HTML 标题是您希望在网页上显示的标题或副标题。

## 实例

```html
标题 1
标题 2
标题 3
标题 4
标题 5
标题 6
```

[亲自试一试](https://www.w3school.com.cn/tiy/t.asp?f=html_headings)

## HTML 标题

HTML 标题使用 `<h1>` 到 `<h6>` 标签定义。

`<h1>` 定义最重要的标题。 `<h6>` 定义最不重要的标题。

### 实例

```html
<h1>标题 1</h1>
<h2>标题 2</h2>
<h3>标题 3</h3>
<h4>标题 4</h4>
<h5>标题 5</h5>
<h6>标题 6</h6>
```

[亲自试一试](https://www.w3school.com.cn/tiy/t.asp?f=html_headings)

注意：浏览器会自动在标题前后添加一些空白（边距）。

## 标题的重要性

搜索引擎使用标题来索引网页的结构和内容。

用户通常通过标题来快速浏览页面。使用标题来展示文档结构非常重要。

`<h1>` 标题应用于主标题，其次是 `<h2>` 标题，然后是较不重要的 `<h3>` ，依此类推。

例如：

### 实例

```html
<h1>旅行指南</h1>

<h2>欧洲</h2>
<h3>法国</h3>
<h3>意大利</h3>

<h2>亚洲</h2>
<h3>印度</h3>
<h3>泰国</h3>
```

[亲自试一试](https://www.w3school.com.cn/tiy/t.asp?f=html_headings_structure)

提示：每页仅使用一个 `<h1>` - 它代表主要主题或标题。

注意：HTML 标题仅用于标题。不要使用标题来使文本 *变大* 或 *加粗* 。

## 更大的标题

每个 HTML 标题都有默认大小。但是，您可以使用 `style` 属性和 CSS `font-size` 属性为任何标题指定大小：

### 实例

```html
<h1 style="font-size:60px;">标题 1</h1>
```
