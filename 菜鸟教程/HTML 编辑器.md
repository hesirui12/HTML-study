学习 HTML 只需要一个简单的文本编辑器。

## 使用记事本或 TextEdit 学习 HTML

网页可以通过专业的 HTML 编辑器来创建和修改。

但是，为了学习 HTML，我们推荐使用简单的文本编辑器，例如记事本（PC）或 TextEdit（Mac）。

我们相信使用简单的文本编辑器是学习 HTML 的好方法。

按照以下步骤使用记事本或 TextEdit 创建你的第一个网页。

### 步骤 1：打开记事本（PC）

#### Windows 8 或更高版本：

打开开始屏幕（屏幕左下角的窗口图标）。输入 "记事本"。

#### Windows 7 或更早版本：

打开 开始 > 程序 > 附件 > 记事本

### 步骤 1：打开 TextEdit（Mac）

打开 Finder > 应用程序 > TextEdit

同时更改一些首选项，以便应用程序能正确保存文件。在 首选项 > 格式 中，选择 "纯文本"。

然后在 "打开和保存" 下，勾选 "将 HTML 文件显示为 HTML 代码而非格式化文本"。

然后打开一个新文档来放置代码。

### 步骤 2：编写一些 HTML

将以下 HTML 代码写入或复制到记事本中：

```html
<!DOCTYPE html>
<html>
<body>

<h1>My First Heading</h1>

<p>My first paragraph.</p>

</body>
</html>
```
![记事本](https://www.w3school.com.cn/i/html/notepad.png)

### 步骤 3：保存 HTML 页面

将文件保存到你的计算机上。在记事本菜单中选择 文件 > 另存为。

将文件命名为 "index.htm"，并将编码设置为 *UTF-8* （这是 HTML 文件的首选编码）。

![保存文件](https://www.w3school.com.cn/i/html/notepad-save.png)

提示：你可以使用.htm 或.html 作为文件扩展名。两者没有区别，完全由你决定。

### 步骤 4：在浏览器中查看 HTML 页面

在你喜欢的浏览器中打开保存的 HTML 文件（双击文件，或右键单击并选择 "打开方式"）。

结果将类似于这样：

![在浏览器中查看](https://www.w3school.com.cn/i/html/chrome.png)
